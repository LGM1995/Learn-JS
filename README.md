## Run 06 Number

## Run 08 Object

## Run 09 undefined and null
### undefined와 null 은 js에서 모두 값이 없음을 뜻한다.
### undefined는 정의되지 않은 변수나 속성이라 볼 수 있지만 null의 경우 특수한 의도에 의해 개발자가 null로 지정한 값일 가능성이 매우 높다.

## Run 10 Arithmatic과 Operator
### 선언된 변수에 산술자를 사용하여 계산이 가능
### ++와 --연산자 사용의 경우 그 위치에 따라 행이 끝나고 실행되거나 연산후에 실행이 된다.
### js에서 Math.pow(a,b); 를 사용하면 a의 b승을 구한다.
### Math.sqrt(b); 를 사용하면 b의 제곱근을 구한다.
### Math.random(); 을 사용하면 호출시 0~1사이의 난수를 가져다 준다.

## Run 11 function(함수)

## Run 12 Relational Opertator
### 관계연산자를 사용하여 값의 차이나 동일 여부를 알 수 있다.
### 문자열을 비교하는 경우 사전순으로 비교한다.
### && 연산 사용시 모두 참이여야 참 || 하나라도 참이면 참이다.

## Run 13 연산
### 산술 연산시 ()안의 값이 먼저 연산되며 곱과나누기가 더하기 빼기보다 먼저 실행된다.
### 연산자 우선순위를 사용하더라도 실수할 수 있으니 괄호를 적극적으로 사용하라.

## Run 14 String 이어붙이기
### 문자열은 length를 사용하여 길이를 구할 수 있다.
### str1.concat(str2);를 사용하면 str1과 str2를 붙여서 사용이 가능하다.
### str1.concat(str2).concat("!"); 등 계속 이어서도 사용이 가능하다.
### +를 사용하여도 붙여 쓸 수 있으며 문자와 숫자를 더하는 경우 숫자를 문자로 자동 변환하여 출력한다.

## Run 15 문자열
### 문자열은 배열상태로 저장되기 때문에 charAt(n)을 사용하면 n번째 위치의 값을 반환한다.
### 비어있는 경우 빈 문자가 출력된다.
### str[n]을 사용하여도 출력이 가능하며 비어있는 경우 undefined를 출력하기 때문에 더 많이 사용한다.
### str.substring(n,m);을 사용하면 n번째 부터 m-1까지 출력된다.
### str.substr(n,m);의 경우 n번째 부터 m개의 문자를 출력한다.
### str.substr();의 경우 n이 음수이면 뒤에서 부터 n번째 부터 출력한다.
### substirng과 substr둘다 m을 생략할 경우 n번째 부터 마지막 까지 출력한다.
### indexOf(x)는 x가 들어간 첫번째 위치를 알려주고 lastindexOf(x)를 사용할 경우 x가 들어간 마지막 위치를 알려준다. 없는 경우에는 -1을 반환한다.

## Run 16 배열
### 자바스크립트의 배열에는 다양한 속성값을 선언이 가능하며 []를 사용하여 속성값을 초기화한다.

## Run 17 배열의 사용
### pop()을 사용하면 배열의 마지막이 빠져나온다. 배열의 원소가 출력되면서 배열에서는 사라진다.
### shift()을 사용하면 배열의 맨 처음이 빠진다.
### push()를 사용하면 배열 마지막에 원소를 추가하고 배열의 길이를 반환한다.
### unshift()를 사용하면 배열 맨 처음에 원소를 추가하고 배열의 길이를 반환한다.
### reverse()를 사용하면 배열의 위치를 정 반대로 전부 바꾸고 배열을 반환한다.
### sort()를 사용하면 배열안의 값들을 차례로 정렬하고 배열을 반환한다.
### 배열과 배열을 concat을 사용하여 배열을 합칠수 있다.
### split(",")를 사용하여 문자열에서 , 를 구분하여 배열로 저장이 가능하다.

## Run 17 주석
### 프로그램의 가동에 있어 전혀 관련없는 코드이나 코드의 설명을 붙이거나 정의를 하기 위해서 개발자들의 코맨트라고 볼 수 있다.
### 개발동안 개발의 편의를 위해 주석처리를 하는 경우도 많다.
### 한줄의 경우 //를 사용하고 여러줄의 주석이 필요한 경우 /* */를 사용하며 각 줄 맨앞에 *를 달수 있어 가독성을 높힐수 있다.
### 단 프로그램 배포 전에 필요없는 주석은 지워야하며 그렇지 못하는 경우 클라이언트에게 필요없는 메시지를 전달할 수도 있다.

## Run 18 if
### if문은 조건을 제시하고 참인 경우에만 프로그램을 실행시킬 수 있다.
### if와 else if를 사용할 시 주의사항으로는 선차 조건식이 참일경우 if문을 종료하기 때문에 후에 나오는 조건식을 실행하지 않는다.
